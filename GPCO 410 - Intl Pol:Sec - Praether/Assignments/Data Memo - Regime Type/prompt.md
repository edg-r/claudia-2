Memos must follow all conventions of attribution and notation. ALL CITATIONS SHOULD BE MADE IN FOOTNOTES. Quotes and paraphrasing must be explicitly cited.

> **Instructions:**
> - **Step 1:** Pick an observation from a widely available dataset used in IR security studies (e.g. COW, UCDP, PRIO, Systemic Peace – which produces Polity IV, etc).
> - **Step 2:** Pick an aspect of that observation to investigate (e.g. the fatality level; the participants in a conflict; democratic institutions etc)
> - **Step 3:** Answer the following questions
>   - What dataset did you choose?
>   - What observation did you choose?
>   - What aspect of the observation are you investigating?
>   - How was the observation originally coded?
>   - Why was it coded that way according to the code book or other materials provided by the provider of the dataset?
>   - Do you find the coding problematic? Why? If not, why not?
>   - If you were to code it differently, why would you code it differently? Can you give an example?
>   - If the coding rules for the aspect you investigated were different, how might this change our understanding of international politics?
