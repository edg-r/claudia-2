# Claudia Course Workspace — GPCO 410 International Politics and Security

This folder is the assistant-side working structure for the course. It is meant to preserve continuity across the 10-week quarter without interfering with the main course materials.

## Purpose

Use this workspace to maintain:
- cumulative course memory
- theory tracking across weeks
- assignment and deliverable continuity
- recurring workflows for readings, lecture synthesis, and study prep

## Folder map

- `COURSE_MEMORY.md` — running memory of the course arc, professor tendencies, repeated themes, and important context
- `THEORIES.md` — cumulative theory tracker with concise, reusable summaries
- `ASSIGNMENTS.md` — assignment tracker and deliverable status
- `WORKFLOWS.md` — repeatable workflows for readings, lecture review, and paper/exam prep
- `SYLLABUS_NOTES.md` — distilled syllabus notes and course mechanics
- `WEEKLY_SYNTHESIS/` — weekly summaries and carry-forward notes
- `READING_NOTES/` — structured notes on readings and required texts
- `CONCEPT_MAPS/` — quick concept sheets / comparison notes

## Operating principle

The files are the memory. The agent is just the interface.
