# Homework 1 - Codex

This folder contains the script-only Codex working files for `Homework 1` in `GPEC 446 - QM3 - Valasquez`.

## Files Included

- `Homework_1.R`: main analysis script.
- `atlas.csv`: source tract-level dataset.
- `Homework_1_Instructions.docx`: assignment prompt.
- `city_poverty_mobility.png`, `urban_density_deciles.png`: exported figure files generated by the script.
- `table_q12.html`, `table_q3.html`, `table_q4_ovb.html`, `table_q5.html`, `table_q79.html`, `table_open_urban_rural_regression.html`, `table_iv_manual_2sls.html`, `table_code_understanding.html`: standalone table exports generated by the script.
- `README.md`: this file.

## Current Workflow

Run the assignment script directly from this folder:

```bash
Rscript Homework_1.R
```

The script loads the data, builds the variables, runs the regressions, renders figures in the R graphics device, saves PNG figure files, and writes standalone HTML table files.

## Question Map

- Q1-Q2: New Orleans descriptive-statistics table, exported as `table_q12.html`.
- Q3: four-city poverty and mobility regressions, with `city_poverty_mobility.png` and `table_q3.html`.
- Q4-Q5: New Orleans omitted-variable-bias and interaction models, exported as `table_q4_ovb.html` and `table_q5.html`.
- Q6: prose-only causal interpretation note in the script comments.
- Q7-Q9: national pooled and commuting-zone fixed-effect regressions, exported as `table_q79.html`.
- Open question: urban/rural regression table and figure, exported as `table_open_urban_rural_regression.html` and `urban_density_deciles.png`.
- IV section: manual first stage, reduced form, and second stage table, exported as `table_iv_manual_2sls.html`.
- Code Understanding Sufficiency Test: glossary table exported as `table_code_understanding.html`.

## Required R Packages

The script loads:

- `dplyr`
- `ggplot2`
- `readr`
- `stargazer`
- `AER`

Install missing packages with:

```r
install.packages(c("dplyr", "ggplot2", "readr", "stargazer", "AER"))
```

## Notes

- There is no R Markdown companion in this folder now.
- Rendered R Markdown artifacts were removed.
- Regression tables are exported with `stargazer`; non-regression tables use a small local HTML writer for matching standalone formatting.
- The manual IV section follows the homework prompt by using fitted values in the second stage rather than `ivreg()`. Its second-stage standard errors are therefore the manual-regression standard errors, not corrected IV standard errors.
