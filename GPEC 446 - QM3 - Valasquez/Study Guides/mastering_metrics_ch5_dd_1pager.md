## BLUF
DiD recovers causal effects from non-experimental panel data by comparing the **change** in outcomes between a treated group and a control group, subtracting out fixed level differences and common time shocks. Its identifying assumption is **common (parallel) trends**: in the absence of treatment, treated and control units would have evolved in parallel. In regression form, the DiD effect is the coefficient on the treatment-by-post interaction (p. 190).

## Context & Scope
- Two running empirical cases: the 1930-31 Caldwell banking crisis (Atlanta Sixth District easy money vs. St. Louis Eighth District tight money), and a state-year panel of MLDA changes and 18-20-year-old death rates, 1970-1983 (pp. 184-198).
- Positions DiD as a tool for when IV and RDD are unavailable, but parallel pre-trends are plausible (p. 182).

## Key Evidence / Claims
- **Four-number DiD** (two groups, two periods): simple (Y_post - Y_pre)_treat - (Y_post - Y_pre)_control. Atlanta Fed liquidity support saved roughly **19 banks** in Mississippi's Sixth District, about 10% of 1930 banks (p. 187).
- **Regression DiD** uses a treat dummy (fixed group differences), a post dummy (common time shocks), and their interaction (the causal effect). With >2 periods/units, regression DiD is more precise; bank-case regression estimate is ~21 banks saved, SE ~11 (pp. 190-191).
- **Multistate panel extension** swaps the treat dummy for state fixed effects and the post dummy for year fixed effects; treatment is measured by LEGAL_st (share of 18-20-year-olds legally drinking). Estimated effect: about **11 extra deaths per 100,000** 18-20-year-olds, ~7 from motor vehicle accidents (p. 197, Table 5.2).
- **Common-trends assumption** is the core identifying restriction. Relax it with **state-specific linear time trends** to allow controlled nonparallel evolution (pp. 197-198, eq. 5.6).

## Implications for Practice
- Use DiD when you have panel data, a clean treatment timing, and pre-period evidence that treated/control trends tracked each other. Always plot the raw pre-trend before running the regression.
- Report both the "four-number" DiD and the regression DiD; they should agree in the 2x2 case and are a useful sanity check. Add state (or unit) FE + year FE when exploiting multi-unit, multi-period variation, and add unit-specific trends as a robustness check.

## Limits / Counterpoints
- Inference is tricky with few clusters and serial correlation (chapter appendix); small-sample DiD estimates (e.g., n=12 banks) are only marginally significant. Parallel trends is an assumption, not a test — pre-trend plots can make it plausible but never prove it.
